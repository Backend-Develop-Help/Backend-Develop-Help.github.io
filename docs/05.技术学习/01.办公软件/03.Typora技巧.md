# Typora 公式

**Typora是一款支持Markdown的编辑器，亲测非常好用。之前发CSDN博客也都是先在Typora上完成，然后直接导入到CSDN。最近在数学公式编辑上遇到了点麻烦，在此总结了常用的公式编辑方法，旨在文章更加的美观规范。**
