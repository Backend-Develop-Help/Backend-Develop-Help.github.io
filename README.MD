<h2 align="center">开发自救手册</h2>

## 如何贡献
fork该仓库  
npm推荐版本>=18  
npm安装
```shell
npm install
```

根据环境选择
```shell
npm run dev

npm run dev:win
```
按照格式进行开发，编译后会自动帮md文档添加title  

开发完毕通过pr形式合并master分支，合并后自动部署
